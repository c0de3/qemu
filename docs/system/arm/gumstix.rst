Gumstix Connex and Verdex (``connex``, ``verdex``)
==================================================

These machines model the Gumstix Connex and Verdex boards.
The Connex has a PXA255 CPU and the Verdex has a PXA270.

Implemented devices:

 * Flash
 * SMC91C111 ethernet
 * Interrupt controller
 * DMA
 * Timer
 * GPIO
 * MMC/SD card
 * FIR
 * LCDC
 * SSP
 * PC Card sockets
 * I2C
 * I2S
